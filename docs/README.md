# Docs

Canonical documentation root for Vireon Network.

Status: Current documentation index / mixed maturity

Start with:

- `DOCUMENTATION_POLICY.md` — precedence, maturity, history, and publication;
- `DOCUMENTATION_INVENTORY.md` — generated inventory of every Markdown file;
- `source-info/README.md` — canonical project source information;
- `release/NETWORK_MATURITY.md` — public status and G0–G4 boundaries.

The remaining folders contain architecture, protocol, tokenomics, API,
security, legal, release, mining, and operator documentation. Each document's
status applies to that document; a Draft product idea does not reduce the status
of an already implemented core rule, and implemented candidate code does not
imply public Mainnet readiness.
